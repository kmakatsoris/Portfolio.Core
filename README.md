### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/kmakatsoris/Portfolio.Core.git
   cd Portfolio.Core
   ```

2. Build the application:
   ```bash
   dotnet build
   ```

3. Start the application:
   ```bash
   dotnet run
   ```

5. Open your browser and navigate to `http://localhost:5123/swagger/index.html`.
